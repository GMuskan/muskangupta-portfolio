Pure and ImpureFunctions

1. Pure functions

Pure function is a function without any side effects. conside the code below to understand it in a more better way.

function updateMyNames(newName)

{

const myNames = ["Muskan", "Sakshi"];

myNames[myNames.length] = newName;

return myNames;

}

console.log(updateMyNames("Neha"));

Output: ['Muskan', 'Sakshi', 'Neha']

Note: In the code above updateMyNames() does not depend on any external code to accomplish it's duties. This makes ist a pure function

2. Impure functions

Impure function is a function that contains one or more side effects.

const myNames = ["Muskan", "Sakshi"];

function updateMyName(newName)

{

const myNames = ["Muskan", "Sakshi"];

myNames.push(newName);

return myNames;

}

updateMyName("Neha");

Output: ['Muskan', 'Sakshi', 'Neha']

Note: In the code above updateMyName() is an impure function because it contains code (myNames) that mutates an external state - which gives updateMyName() some side effects
