What is Function Currying ?

We simply wrap function inside a function, which means we are going to return a function from another function to obtain some kind of translation. The parent function takes the first provided argument and returns the function that takes the next argument and this keeps on repeating till the number of arguments ends. Hopefully, the function that receives the last argument returns the expected result.

There are 2 ways of currying a function.

1. Using bind()

let multiply = function(x, y){

console.log(x*y);

}

let multiplybyTwo = multiply.bind(this, 2);

multiplyByTwo(5);

bind() method will return a copy of multiply method, replacing x by 2 as shown below:

let multiplyByTwo = function(y){

let x = 2;

console.log(x*y);

}

2. using function closure.

let multiplyFunction = function(x){

return function(y){

console.log("multiplication of" + x "and" + y + "is" + x*y);

}

}

let multiplyByFour = multiplyFunction(4);

multiplyByFour(5);

Advantages of Currying:

  • Currying will make your code easier to refactor.
  • Currying also creates a more declarative code base, e.g. it's easier to read the code and understand what it's doing
  • It helps to create a higher-order function.
  • Currying provides a way for working with functions that take multiple arguments, and using them in frameworks where functions might take only one argument.
